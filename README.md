# L'Élite Gastronomique 🍽️

> "The pinnacle of fine dining. Where every meal is a story and every guest is a legend."

Welcome to the official repository for **L'Élite**, a high-end restaurant web application designed to provide a luxury digital experience that matches the excellence of a world-class kitchen.

## ✨ Project Overview
L'Élite is a modern, full-stack web platform built to bridge the gap between gourmet cuisine and digital convenience. It focuses on immersive storytelling, high-performance UI, and a seamless reservation flow for elite clientele.

## 💎 Elite 0.001% Features
- **Inertial Smooth Scrolling**: Powered by **Lenis** for organic, high-end motion.
- **Magnetic Interaction System**: Custom cursor follower and magnetic UI components for tactile feedback.
- **Horizontal Scroll Gallery**: An immersive visual journey using **GSAP ScrollTrigger** pinning.
- **Cinematic Reveal Masks**: Images reveal through an obsidian mask on scroll for high-impact storytelling.
- **Split-Text Typography**: Hero headline uses nested span animations for a high-impact, branded entry.
- **Filmic Grain Overlay**: Subtle noise texture for a premium, non-digital aesthetic.
- **Fluid Typography & Spacing**: Fully responsive design using modern `clamp()` and `vw` units.

## 🚀 Tech Stack
- **Backend**: Python (Flask)
- **Frontend**: HTML5, CSS3 (Modern Design System)
- **Animations**: GSAP, AOS, Lenis (Smooth Scroll)
- **Assets**: Custom AI-Generated High-Resolution Photography

## 🛠️ Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/VedantNair14/L-Elite.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```

The design follows a **"Cyber-Luxury"** aesthetic—utilizing deep obsidian tones, brushed gold accents, and expansive whitespace to evoke a sense of exclusivity and timeless elegance.
